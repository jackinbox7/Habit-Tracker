# Habit-Tracker

Description
Habit Tracker is a simple Python project designed to help build and maintain daily habits. The program allows you to add habits, mark them as completed, and view your progress in a straightforward way. 

Features
•	Add new habits to track
•	Mark habits as completed each day
•	Display a list of all habits with their status
•	Track the date when a habit was last completed

Technologies/Tools Used
•	Python 3
•	Standard Python libraries (datetime)
•	Command line interface for interaction
